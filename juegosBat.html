¿Qué aprenderás?

  • Pedir datos con set /p
  • Cálculos y uso de %random%
  • Bucles con :etiqueta y goto
  • Validar números con for /f

Preparación (1 min)

  1. Abre el Bloc de notas o VS Code.
  2. Crea un archivo llamado adivina.bat.
  3. Copia el código base que verás abajo.
  4. Guarda y haz doble clic para jugar.

01. Código Base

adivina.bat 
@echo off
title Adivina el numero (BAT)
chcp 65001 >nul :: Soporte para tildes
setlocal EnableExtensions EnableDelayedExpansion

:inicio
cls
echo ============================
echo JUEGO: ADIVINA EL NUMERO (1-20)
echo ============================

:: Crear número secreto
set /a objetivo=(%random% %% 20) + 1
set /a intentos=0

:preguntar
set /p n=Escribe un numero (1-20):

:: Validación de entrada
for /f "delims=0123456789" %%A in ("%n%") do (
    echo Eso no es un numero.
    goto preguntar
)

Descargar Mantenimiento.bat

Cómo funciona la lógica

El Azar

%random% genera un número entre 0 y 32767. Usamos módulo 20 para limitarlo.

El Bucle

La etiqueta :preguntar permite repetir el código hasta acertar.

Validación

Filtramos letras para que el programa no se cierre inesperadamente.

Entrada ¿Pasa? Resultado
12 Sí Sigue al rango 1-20
12a No Error: "Eso no es un número"
hola No Muestra aviso y repite

Versión BONUS: CHOICE

¿Quieres algo más controlado? Usa el comando choice para que el usuario solo pueda pulsar teclas específicas (1-5).

choice /c 12345 /n /m "Pulsa 1-5: "
