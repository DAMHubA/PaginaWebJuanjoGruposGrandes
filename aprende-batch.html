Domina la Terminal

Curso completo: desde los fundamentos de MS-DOS hasta la creación de scripts de mantenimiento avanzados.

1. Fundamentos de Batch

Aprende los conceptos básicos, variables, control de flujo y gestión de procesos.

Lección 1.1 - 1.4

Historia y Contexto: MS-DOS y Windows

Antes de Windows con ventanas, existía MS-DOS (Microsoft Disk Operating System). Aunque hoy usamos interfaces gráficas, el intérprete de comandos (CMD) sigue siendo una herramienta poderosa heredada de esa época.

Un archivo Batch (.bat) es un script que ejecuta comandos de MS-DOS secuencialmente, permitiendo automatizar tareas complejas en Windows.

Lección 2.4 - 2.6

Comandos Básicos, Variables e Interacción

Todo script profesional comienza con @echo off para limpiar la salida visual. Usamos set /p para interactuar con el usuario.

@echo off title Mi Primer Script Interactivo cls echo === BIENVENIDO AL SISTEMA === echo. :: Interacción con el usuario (Lección 2.6) set /p nombre=Por favor, introduce tu nombre: echo Hola, %nombre%, preparando el entorno... :: Pausa para que el usuario pueda leer timeout /t 3
Lección 2.10

Redirección a Archivos y Systeminfo

Podemos guardar el resultado de un comando en un archivo de texto usando los operadores de redirección > (sobrescribir) y >> (añadir al final).

@echo off echo Generando reporte del sistema... :: Guardar información del sistema en un log systeminfo > reporte_pc.txt echo Añadiendo fecha y hora al reporte... echo Reporte generado el: %date% a las %time% >> reporte_pc.txt echo Reporte completado.
Lección 2.11

Control de Procesos: Start y Timeout

Aprende a lanzar procesos en segundo plano y manejar tiempos de espera.

  • timeout /t 10: Espera 10 segundos antes de continuar.
  • start /b: Inicia una aplicación sin abrir una ventana nueva de CMD.
@echo off echo Iniciando calculadora en segundo plano... start /b calc.exe echo Esperando 5 segundos antes de cerrar... timeout /t 5 /nobreak

2. Scripts de Mantenimiento

Automatiza la limpieza, seguridad y reparación de tu sistema Windows.

Lección 2.7 - Limpieza

Limpiar Archivos Temporales

Elimina recursivamente archivos basura para liberar espacio.

:: Borra archivos en subcarpetas (/s) sin preguntar (/q) del /s /q "%temp%\*.*" :: Intenta borrar carpetas vacías rd /s /q "%temp%" md "%temp%"
Lección 2.8 - Red

Optimizar Red (Flush DNS)

Resuelve problemas de conectividad renovando la IP y limpiando el DNS.

ipconfig /release ipconfig /flushdns ipconfig /renew echo Configuración de red restablecida.
Lección 2.11 - Seguridad

Seguridad: Escaneo con Windows Defender

Batch permite invocar herramientas del sistema como el antivirus Windows Defender mediante MpCmdRun.exe.

@echo off echo Iniciando escaneo rápido de antivirus... :: Ejecuta el escáner de Windows Defender "C:\Program Files\Windows Defender\MpCmdRun.exe" -Scan -ScanType 1 if %errorlevel% == 0 ( echo Escaneo completado sin amenazas. ) else ( echo ¡ATENCIÓN! Se encontraron amenazas o hubo un error. ) pause
Lección 2.9 y 2.13 - Sistema

Mantenimiento y Manejo de Errores

Script avanzado que desfragmenta el disco y verifica si la operación fue exitosa usando errorlevel.

@echo off title Optimizador de Disco echo Analizando y desfragmentando C:... :: Defrag en modo detallado (/V) defrag C: /V :: Comprobar código de salida (Lección 2.13) if %errorlevel% NEQ 0 ( echo Error: No se pudo completar la desfragmentación. echo Asegúrate de ejecutar como Administrador. ) else ( echo Mantenimiento finalizado correctamente. ) pause
Lección 2.12 - Backup

Copia de Seguridad Profesional (XCOPY)

Respalda documentos a una unidad externa, copiando solo lo nuevo.

:: /E (subcarpetas) /D (solo nuevos) /C (ignorar errores) /Y (no preguntar) xcopy "C:\Usuarios\User\Mis Documentos" "D:\Backups\Docs" /E /D /C /Y /I
